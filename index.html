Data Analysis @ Horeb Energy Formula UFMG

Advanced telemetry data analysis platform for Formula Student competitions

Application Status

This is a Flask web application that requires a server to run. GitHub Pages serves static files only.

How to Access the Application

For Development:

  1. Clone this repository
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: python flask_app.py
  4. Access at: http://localhost:5000
